# ACGN收藏管理工具

一个基于Bangumi的ACGN收藏管理工具

## 主要功能
- Bangumi账号同步收藏
- 多站点视频搜索和播放
- 本地数据库缓存
- PikPak下载
- <details>
    <summary> <b>未来计划</b> </summary>
  
  * 视频本地缓存
  * 普通BT下载
  * 边下边看
  * 弹幕 
  * 超分辨率
  * 规则编辑器
  * 规则自定义
  * 规则排序
  * 记录播放历史
  * 剧时间表
  * 漫画.轻小说更新提醒
  * 漫画.轻小说源
  * GalGame下载.启动.统计
  * 多端适配
  </details>

## 技术
- C++ 17+
- UI 使用 [Qt 6.10.2](https://doc.qt.io/qt-6)
- XPath选择器 使用 [libxml2](https://github.com/GNOME/libxml2)
- 视频播放器 使用 [VLC](https://www.videolan.org/vlc)
- 验证码处理 使用 [ddddocr](https://github.com/sml2h3/ddddocr)
